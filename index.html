Expand your streams.

This is a fork of StreamControl to make it more suited for SC2 tournaments and other games, it will still retain the fighting game support of the original release.

StreamControl writes an XML file for use with XSplit and it's ability to use SWF files to display information. It has support for many purposes but mostly gaming.

If you have no idea how to edit the flash files it is also possible to use StreamControl with the default XSplit title plugins. Just point it at the streamcontrol.xml file in your XSplit directory and you'll be good to go.

Rough how-to: 1. Place StreamControl.exe in it's own directory (shouldn't actually matter where) 2. Run it. It should autodetect your XSplit installation if you installed XSplit to the default location. If not, please click configure and set it there. 3. Load XSplit and set up the scene. Add the SWF overlays you require. 4. Done.

Report bugs at http://github.com/Soche/StreamControl/issues

List of variables pluggable for swf's.
Description:     Variable:
Player 1 name  - pName1
Player 2 name  - pName 2
Player 1 color - p1c
Player 2 color - p2c
Player 1 score - pScore1
Player 2 score - pScore2
# of rounds    - rounds
Special round  - srText
Caster left    - cTitle1
Caster right   - cTitle2
Title left     - mText1
Title right    - mText2
Twitter left   - mText3
Twitter right  - mText4
Name of game   - game
Timestamp      - timestamp
